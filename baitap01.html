Carousel hình ảnh

Giải thích logic tư duy:

1. Xử lý index để tránh ngoài giới hạn:

- Sử dụng công thức: index = (index + step + total) % total

- Khi index âm: cộng total để đưa về dương, rồi chia dư để quay vòng

- Khi index vượt quá: chia dư để quay về đầu

- Ví dụ: index = -1 → (-1 + 6 + 6) % 6 = 5 (quay về slide cuối)

2. Tối ưu performance:

- Chỉ thay đổi transform: translateX() thay vì thay đổi layout

- Sử dụng will-change: transform để GPU acceleration

- Tự động pause khi hover để tiết kiệm tài nguyên

- Sử dụng requestAnimationFrame cho animation mượt mà
